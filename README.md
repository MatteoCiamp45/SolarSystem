## Descrizione

Simulazione interattiva del sistema solare realizzata in WebGL 1.0 e JavaScript puro.

Include:
- Shader GLSL personalizzati
- Caricamento mesh OBJ/MTL
- Orbite planetarie simulate
- Camera interattiva
- Skybox e illuminazione solare
